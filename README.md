# coding_test

알고리즘과 자료구조 문제 풀이를 기록하는 개인 워크스페이스입니다.

## 디렉터리 개요
- `doit_coding_test/`: Do it! 알고리즘 코딩 테스트(파이썬편) 문제 풀이와 설명을 정리한 공간
- 그 외 카테고리별 폴더: 진행 상황에 따라 추가되는 연습 코드 모음

## 작업 흐름
- 원격 저장소는 `work` 브랜치만 사용하며 **새 브랜치 생성이 제한**되어 있습니다.
- 로컬에서 작업할 때는 아래 순서로 진행하면 PR 생성 시 오류를 피할 수 있습니다.
  1. `git checkout work`로 기본 작업 브랜치에 있는지 확인합니다.
  2. 변경 사항을 커밋합니다.
  3. `make_pr` 도구를 호출해 PR을 생성합니다. (브랜치를 추가로 만들 필요가 없습니다.)
- 여전히 "브랜치 생성이 허용되지 않았다"는 메시지가 나온다면, 원격 저장소에 대한 **Write 권한**이 있는지 확인하고, 필요 시 저장소 관리자에게 권한 추가 또는 포크 사용을 요청하세요.

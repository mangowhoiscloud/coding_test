# BOJ 1874 - 스택 수열

- 문제 링크: [https://www.acmicpc.net/problem/1874](https://www.acmicpc.net/problem/1874)
- 해설: 1부터 N까지의 수를 순서대로 스택에 push하면서 목표 수열과 일치할 때까지 pop을 수행한다. push/pop 연산의 기록으로 수열 생성 가능 여부와 연산 순서를 출력하고, 중간에 만들 수 없으면 NO를 출력한다.
- 실행 결과:
  ```text
  입력
  8
  4
  3
  6
  8
  7
  5
  2
  1

  출력
  +
  +
  +
  +
  -
  -
  +
  +
  -
  +
  +
  -
  -
  -
  -
  -
  -
  ```
